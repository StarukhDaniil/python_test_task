Код створює папку csv в директорії, з якої він запущений(якщо її немає)

Основна логіка коду:
    - створюється додатковий процес, який буде відправляти дані
    - симулятор спочатку зчитує повідомлення з .tlog, потім пакує його в сирі байти(щоб передавати менше байтів)
    - дані передаються по TCP
    - хендлер читає сирі байти, та поступово логує повідомлення в .csv файли в папці csv/
